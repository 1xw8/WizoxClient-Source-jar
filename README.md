## 📦 Installation

Wizox Client includes a **pre-built `.jar` file** for anyone who does not want to build the source code themselves.

### Option 1 — Use the built JAR
1. Download the included `.jar`
2. Place it into your Minecraft setup
3. Launch and enjoy Wizox Client

### Option 2 — Build from source
1. Clone the repository
2. Open the project
3. Build the client yourself
4. Use the generated JAR
